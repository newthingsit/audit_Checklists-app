# QA Audit - CDR Plan Template

## Overview

The QA Audit - CDR Plan template is a comprehensive checklist template designed for Quality Assurance audits focusing on Customer Data Record (CDR) compliance and quality assurance processes.

## Template Details

- **Name**: QA Audit - CDR Plan
- **Category**: Quality Assurance
- **Description**: Comprehensive QA Audit checklist for Customer Data Record (CDR) compliance and quality assurance
- **Total Items**: 24 checklist items

## Installation

### Using the Script

Run the provided script to create the template in your database:

```bash
cd backend
node scripts/create-qa-audit-template.js
```

The script will:
1. Check if the template already exists
2. Create the template if it doesn't exist
3. Add all 24 checklist items
4. Display success message

### Manual Creation

You can also create the template manually through the web interface:
1. Go to Checklists page
2. Click "Create Template"
3. Enter template details
4. Add checklist items one by one

## Usage

Once created, the template can be used to:
- Create new audits using this template
- Schedule recurring audits
- Generate audit reports
- Track compliance over time

## Customization

You can customize the template by:
- Editing item descriptions
- Adding new items
- Modifying categories
- Adjusting required status
